# Lottie to Webp (animated)

Basic client sided tool to convert Lottie animations to Webp format locally.

### Main Libraries used:

- [Lottie-Web](https://github.com/airbnb/lottie-web)
- [Webp.wasm](https://github.com/nieyuyao/webp-wasm)
