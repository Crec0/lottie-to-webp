# Lottie to Webp (animated)

Basic client sided tool to convert Lottie animations to Webp format locally.

### Main Libraries used:

- [Lottie](https://github.com/airbnb/lottie-web)
- [Wasm Webp](https://github.com/nieyuyao/webp-wasm)
